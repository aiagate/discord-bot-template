# app

このフォルダはアプリケーションの主要なロジックを含みます。

- `domain/`: ドメインモデルやビジネスルールを定義するフォルダ。フレームワーク非依存。
- `usecases/`: ユースケース（アプリケーションの具体的な操作）を定義するフォルダ。
- `infrastructure/`: データベース、外部API、メッセージングなどの技術的詳細を実装するフォルダ。
- `presentation/`: 外部インターフェース（API, Discord Bot, Worker）を定義するフォルダ。
- `container.py`: 依存性注入（DI）の設定。
- `__init__.py`: パッケージの初期化。
